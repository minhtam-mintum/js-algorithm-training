# JavaScript Algorithm Training

A dependency-free workspace for learning algorithms with JavaScript and
`console.log`.

## Run

```bash
cd /home/minhtam-mintum/Works/js-algorithm-training
npm start
```

Edit `src/index.js`, save it, and run `npm start` again.

## First exercise

The starter implements binary search and logs every search step. Try:

1. Changing `target`.
2. Returning how many steps the search used.
3. Rewriting it recursively.
4. Handling an array sorted in descending order.
